📜 CSV UPLOAD GUIDE ×

วิธีใช้งาน

1. คลิกปุ่ม UPLOAD CSV (มุมขวาบน หรือกล่องด้านล่าง Sidebar)

2. เลือกไฟล์ .csv ที่ต้องการ

3. Dashboard จะอัปเดตอัตโนมัติ — ตัวเลข, กราฟ, ตาราง, ฟิลเตอร์ทุกอย่าง

คอลัมน์ที่ระบบรองรับ (header แถวแรกของ CSV)

No — ลำดับ
year — ปี (2023–2026)
Month — YYYY-MM-DD
Quarter — Q1–Q4
Client — แบรนด์ลูกค้า
Client Type — New / Old / New Retain / Old Upsize / Internal
Project — ชื่อโปรเจ็ค
Work Tpye — ประเภทงาน
Responsibility — เจ้าของงาน
Sale Lead — เซลล์ที่หางาน
Brief Date — YYYY-MM-DD
Proposal Date — YYYY-MM-DD
Budget — ตัวเลข (มี comma ได้)
Industries — อุตสาหกรรม
Presentation — Link Plan
Result — Approve / Not Approve / Cancel / Wait Feedback / Proposal
Reason — เหตุผล
💡 Tip: ใช้ Google Sheets export เป็น .csv ได้เลย — File → Download → Comma-separated values (.csv)
ถ้าหัวคอลัมน์เป็น Work Type หรือ Year ก็รองรับเหมือนกัน ระบบจะแปลงให้อัตโนมัติ
